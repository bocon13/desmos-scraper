# Desmos Activity Scraper

Requirements:
- node

To install:
```bash
git clone https://github.com/bocon13/desmos-scraper.git
cd desmos-scraper
npm install
```

To run:
```bash
node scrape.js
```

Example output:
```

  'Teacher Account'
]
```
